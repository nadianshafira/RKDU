Analisis pada penelitian ini menggunakan regresi data panel metode Fixed Effect Model dengan estimasi parameternya menggunakan Least Square Dummy Variable (LSDV). Hasil analisis menunjukkan bahwa hanya variabel total asset yang berpengaruh secara signifikan terhadap Return on Investment Total (mencakup pendapatan yang sudah dan belum direalisasikan) pada lembaga dana pensiun di Provinsi DKI Jakarta dan Jawa Barat.
